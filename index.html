Race and Income

X-axis: Median household income

Each dots horizontal position represents the estimated income for that census tract (from $20,000 to $250,000 in the example).

Y-axis: Largest group in Census tract

This is a categorical axis showing the racial group that is the largest in each tract (White, Hispanic, Black, or Asian).

Each row corresponds to one of these groups.

Facets (Small Multiples): Each panel (strip) represents a borough of NYC:

Bronx, Brooklyn, Manhattan, Queens, Staten Island

So youre seeing the same kind of plot repeated for each borough, which helps compare across them.
